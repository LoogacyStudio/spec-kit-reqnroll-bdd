# Contributing

## Scope

This extension focuses on Reqnroll BDD planning, Gherkin generation, task injection, implementation handoff, and verification for Spec Kit.

## Non-goals

- No custom implementation command.
- No automatic production code generation.
- No Godot UI automation.
- No automatic NuGet installation.

## Development Checklist

- Update command docs.
- Update artifact templates.
- Update README when workflow changes.
- Update CHANGELOG.
- Run dogfood flow before release.
